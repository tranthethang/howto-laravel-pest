__1.__ Laravel IDE Helper
> https://github.com/barryvdh/laravel-ide-helper

```shell
# PHPDoc generation for Laravel Facades
php artisan ide-helper:generate
```

```shell
# PHPDocs for models
php artisan ide-helper:models
```

```shell
# PhpStorm Meta file
php artisan ide-helper:meta
```
